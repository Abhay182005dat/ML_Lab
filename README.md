## ML Lab
 
This repository contains machine-learning notebook experiments from my ML Lab at NIE.

## Notebooks included  
- `Lab1.ipynb` – first ML exercises  
- `End_to_End_ML.ipynb` – full workflow: data → model → evaluation  
- `MNIST.ipynb` – digit classification with MNIST  
- `NaiveBayes_EmailSpam.ipynb` – spam detection using Naive Bayes  
- `Naive_Bayes.ipynb` – Naive Bayes classification examples  
- `Regression_Training_&_Evaluation.ipynb` – regression tasks with evaluation metrics
